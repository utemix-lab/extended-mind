# SYSTEM FUNCTIONS

Каталог функций без привязки к реализации.  
Каждая функция описывается с трёх перспектив: [VISITOR] / [LAB] / [MEDIA].

## Navigation & Modes
- Query Mode
  - [VISITOR] быстро найти релевантные элементы по тегу
  - [LAB] тестировать поведение контекстов и фильтрации
  - [MEDIA] фиксировать “как мы нашли X”
- Inspect / Report (future)
  - [VISITOR] видеть объяснение результата
  - [LAB] собирать отчёты для проверки качества
  - [MEDIA] формировать краткие нарративы

## Canon
- Узлы / связи / виды
  - [VISITOR] ориентироваться в структуре смысла
  - [LAB] держать канон минимальным и стабильным
  - [MEDIA] создавать системные narrative‑узлы

## Catalogs & Exports
- JSONL каталоги
  - [VISITOR] видеть варианты без перегруза графа
  - [LAB] подключать большие данные без роста канона
  - [MEDIA] собирать “доказательства” через внешние ссылки
- Регистры (tags/country)

## Context Stack
- Проекции (`cap:`, `provider:`, `country:`, `domain:`)
- Refine (entity filters)
  - [VISITOR] уточнять без смены маршрута
  - [LAB] сохранять логику контекстов отделённой от канона
  - [MEDIA] фиксировать цепочки уточнений

## Characters
- Дайджесты, роли, голоса
  - [VISITOR] видеть перспективы, а не только факты
  - [LAB] разворачивать истории без роста каталога
  - [MEDIA] создавать устойчивые narrative‑голоса

## Practices
- Структурный режим действий
  - [VISITOR] понимать “как действовать”
  - [LAB] связывать практики с узлами без дублирования
  - [MEDIA] оформлять маршруты как истории

## Story‑nodes / ADR / Signatures
- Нарративный слой решений
  - [VISITOR] получать объяснения системы
  - [LAB] фиксировать траекторию решений
  - [MEDIA] производить “истории развития”
- Self‑description outputs (profiles / briefs / tutorials)
  - [VISITOR] получать понятные объяснения “что это”
  - [LAB] фиксировать смысл изменений в одном месте
  - [MEDIA] быстро формировать материалы для разных аудиторий
  - Output formats (baseline):
    - One‑paragraph overview (consumer)
    - 5‑bullet brief (partner)
    - 3‑slide summary (investor)
    - Tutorial outline (user)

## Assets
- Виджеты, флаги, бренды
  - [VISITOR] визуальные якоря и контекст
  - [LAB] ассеты не превращаются в сущности
  - [MEDIA] визуальная сцена поддерживает нарратив

## RAG hooks
- Требования к будущей индексации
  - [VISITOR] получать обогащённые ответы без смены модели
  - [LAB] подключать RAG без перестройки канона
  - [MEDIA] автоматически собирать резюме
