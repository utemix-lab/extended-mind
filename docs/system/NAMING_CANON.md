# Naming Canon v1

Единый канонический список правильных написаний.  
Каждое имя/название имеет **ровно две формы**: системную и нарративную.  
Документ пополняемый.

## Principles

- **Системное имя** используется в каноне, идентификаторах, файлах, контрактах.
- **Нарративное имя** используется в сценах, витринах и коммуникации.
- **Других написаний быть не должно.**
