# CHARACTERS

Список персонажей системы и их роль.

## Формат (минимум)

- **ID**
- **Имя**
- **Фокус**
- **Практики**
- **Проекции** (внутр./внешн.)
- **Workbench’и** (процессы)

## Персонажи (черновик)

### character‑vova
- **Имя:** Вова
- **Фокус:** музыкальная культура + инструменты создания звука
- **Практики:** кураторство, исследование, звуковое мышление
- **Проекции:** public feed; workshop space
- **Workbench’и:** `docs/workbenches/wb-vova-music-general.md`, `docs/workbenches/wb-vova-vstablishment.md`, `docs/workbenches/wb-vova-ai-music-signals.md`

### character‑petrova
- **Имя:** Петрова
- **Фокус:** визуальная культура, анимация, брендинг
- **Практики:** режиссура, визуальная навигация
- **Проекции:** story‑сцены, внешние площадки
- **Workbench’и:** обзор сигналов, разбор инструментов, сбор маршрутов
