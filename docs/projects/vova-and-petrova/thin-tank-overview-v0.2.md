# Thin-Tank — Overview (v0.1)

## 1. Что такое thin-tank

**Thin-tank** — это локальный механизм проекта Vova i Petrova,  
предназначенный для автоматизации сбора, структурирования  
и навигации по контенту и знаниям проекта.

Он является:
- прикладным,
- камерным,
- адаптированным под один проект.

---

## 2. Задачи thin-tank

Thin-tank решает следующие задачи:

- сбор публикуемого контента;
- связывание контента с темами, тегами и ролями;
- формирование базы знаний на сайте;
- сокращение ручной работы по структурированию.

---

## 3. Связь с extended-mind

- thin-tank — **дочерний механизм** extended-mind;
- использует:
  - принципы,
  - паттерны,
  - архитектурные решения extended-mind;
- не повторяет его функций.

Extended-mind:
- мыслит,
- проектирует,
- удерживает контекст.

Thin-tank:
- применяет,
- агрегирует,
- автоматизирует в рамках одного проекта.

---

## 4. Границы thin-tank

### Thin-tank НЕ:
- универсальная система мышления;
- замена extended-mind;
- исследовательская платформа.

### Thin-tank ЯВЛЯЕТСЯ:
- инструментом продакшена;
- механизмом демонстрации возможностей;
- “маленьким мозгом” проекта Vova i Petrova.

---

## 5. Статус

Этот документ фиксирует thin-tank как принцип и механизм,  
а не его техническую реализацию.

Реализация может меняться  
без изменения extended-mind.
