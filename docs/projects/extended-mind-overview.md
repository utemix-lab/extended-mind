# Project Overview — Extended-mind  
Версия 0.1

---

## 1. Миссия проекта

Extended-mind — мозг и каркас utemix-lab.  
Он фиксирует структуру смыслов, формирует культуру мышления и служит учителем-архитектором.

---

## 2. Границы

**Входит:**
- система мышления,  
- онтология,  
- принципы,  
- слои,  
- ADR,  
- шаблоны,  
- RAG-интеграция.

**Не входит:**
- финальные художественные проекты,  
- публичный контент,
- техническая реализация сайта.

---

## 3. Основные компоненты

- Definition Pack  
- System Prompt  
- Core Principles  
- ADR  
- Patterns  
- Описание слоёв  
- Процесс трансформации материалов

---

## 4. Взаимосвязи

Extended-mind обеспечивает структуру:
- `Vova & Petrova`  
- `evoquant`  
- `word-machine`  
- будущих проектов

---

## 5. Статус

Версия 0.1 — foundational.
