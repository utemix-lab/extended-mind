# story-node-012

## Название
Тег как якорь запроса

## Что произошло
Query Mode для домена AI стал дефолтным состоянием, а теги превратились в основной вход к каталогу.

## Почему это было необходимо
Текстовые подсказки создавали ощущение “страницы”, а не режима ориентации. Нужен был чистый механизм запроса без onboarding.

## Принятое решение
Сделать Query Mode дефолтом для domain-ai и оставить только pills‑теги как элементы управления.

## Что изменилось в системе
- Теги стали единственным интерфейсом запроса.
- Выдача всегда живёт в Service как режим, а не как переход.
- Канон остался минимальным, данные вынесены в каталоги.

## Открытый вопрос
Когда и как вводить многотеговые запросы, не превращая UI в фильтровую панель?

## Что это закладывает
Переход от “страниц домена” к режимам навигации по океану данных.
