# story-node-031

## Название
Лицензионный слой стал поддерживаемым и видимым

## Что произошло
Слой лицензирования и атрибуции получил полный набор текстов лицензий, явную фиксацию в README и минимальный скрипт аудита.

## Почему это было необходимо
Открытость проекта требовала не только юридических текстов, но и регулярного механизма поддержки актуальности слоя.

## Принятое решение
Закрепить лицензии и правила в репозитории, сделать их видимыми в README и добавить лёгкий audit‑скрипт.
