# Pattern: Thesis → Line  
Версия 0.1

---

## 1. Исходная ситуация

Имеются несколько тезисов:
- оформленных,
- коротких,
- структурных,
- относящихся к общему смысловому полю.

Нужно понять, формируют ли они линию.

---

## 2. Шаги преобразования

### Шаг 1 — Выявить общее направление
Extended-mind определяет:
> “Какой вопрос или тема объединяет эти тезисы?”

### Шаг 2 — Проверить непротиворечивость
Линия должна быть:
- непротиворечивой,
- однородной по уровню абстракции.

### Шаг 3 — Определить функцию линии
Линия может быть:
- исследовательской,
- эстетической,
- инженерной,
- метафилософской.

### Шаг 4 — Оформить линию
Extended-mind создаёт документ:
- название,
- миссия,
- перечень тезисов,
- возможные направления развития.

---

## 3. Результат

Группа тезисов превращается в устойчивое направление мышления.  
Линии становятся материалом для проектов.
