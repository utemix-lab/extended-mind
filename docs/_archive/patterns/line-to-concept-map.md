# Pattern: Line → Concept Map  
Версия 0.1

---

## 1. Исходная ситуация

Есть оформленная линия:  
- тема,  
- набор тезисов,  
- направления развития.

Нужно превратить её в карту смыслов.

---

## 2. Процесс

### Шаг 1 — Определить ключевые узлы
Каждый тезис = потенциальный узел.

### Шаг 2 — Определить тип связи
Связи могут быть:
- причинные,  
- иерархические,  
- ассоциативные,  
- технологические,  
- философские.

### Шаг 3 — Добавить контекст
Extended-mind сопоставляет:
- проекты,  
- персонажей,  
- ADR.

### Шаг 4 — Построить карту
Карта оформляется как документ:
- узлы,
- связи,
- направления вывода.

---

## 3. Результат

Линии обретают форму, пригодную для:
- архитектурных решений,
- метапроектов,
- RAG-поиска,
- объяснения внешнему наблюдателю.
