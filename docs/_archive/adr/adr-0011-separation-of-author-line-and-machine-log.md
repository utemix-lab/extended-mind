# ADR-0011  
**Separation of Author-line and Machine-log**

---

## 1. Контекст

Story-layer состоит из:

- **author-line** — художественная, субъективная, эмоциональная линия автора;  
- **machine-log** — последовательная хроника инженерных решений.

В реальности они часто переплетаются, и extended-mind должен избегать смешения.

---

## 2. Решение

**Author-line и machine-log — два разных типа story-node.**

Их нужно разделять:

- по функциям,
- по тону,
- по формату.

### Author-line  
художественный поток + субъективность → материал для образов.

### Machine-log  
хронология решений → инженерный след.

---

## 3. Обоснование

- художественный слой не должен искажать историю решений;  
- инженерный слой не должен попадать в художественный нарратив;  
- RAG нуждается в чистоте слоёв.

---

## 4. Последствия

Плюсы:
- легче структурировать story-layer;  
- можно выводить разные ветви контента;  
- машина не путается, что является аргументом, а что — образом.

Минусы:
- автор должен выделять тип story-node (или машина — по правилу).

---

## 5. Статус

**Принято.**
