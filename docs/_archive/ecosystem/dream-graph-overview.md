# dream-graph — визуальный рендер экосистемы

> **dream-graph** — рендер со вкладками разных сцен и стеков.  
> Среда для поиска визуальных выразительных средств и элементов пользовательского UI.

## Что это

dream-graph — это **визуальный рендер графов**, не редактор.

Он превращает абстрактный JSON-граф в пространственный, живой объект.

## Роль в экосистеме

```
extended-mind-console (редакторы)
        ↓ экспорт Cosmos Map
dream-graph (рендер 3D)
        ↑
vovaipetrova-site (классические UI-элементы)
```

| Инструмент | Что делает | Данные |
|------------|------------|--------|
| **extended-mind-console** | Редактирует | Сохраняет сессии |
| **dream-graph** | Рендерит 3D | Только читает |
| **vovaipetrova-site** | Классический UI | Экспортирует компоненты |

## Сцены (вкладки)

dream-graph — это **платформа со вкладками разных сцен**.

Каждая сцена — эксперимент с визуальной формой:

| Сцена | Описание |
|-------|----------|
| **Planetary** | Орбитальная модель с in-world панелью |
| **Mono Graph** | Минималистичная монохромная сцена |
| **Universe Map** | Экосистемный вид (Universe Graph) |
| **Core Taxonomy** | Граф таксономии vovaipetrova-core |
| **Legacy** | Зафиксированная сцена v0.1 |

URL-параметры: `?scene=planetary`, `?scene=mono`, `?scene=universe`, `?scene=core-taxonomy`

## Что делает dream-graph

✅ **Делает:**
- Рендерит графы из разных источников
- Экспериментирует с визуальными формами (3D, физика, звук)
- Экспортирует данные для построения графов
- Импортирует элементы классического UI из vovaipetrova-site
- Поиск UI-решений для будущих продуктов

❌ **Не делает:**
- Не редактирует данные
- Не является источником истины
- Не пишет данные обратно

## Источники данных

dream-graph может загружать графы из:
- Локальных JSON-файлов (drag & drop)
- URL (включая GitHub raw)
- Cosmos Map из extended-mind
- Экспортов vovaipetrova-core

## Контракт данных: Cosmos Map

Формат входных данных: **Cosmos Map** (`schema.v0.2`).

```json
{
  "schema": "cosmos-map.v0.2",
  "nodes": [
    { "id": "node-1", "label": "Example", "type": "project" }
  ],
  "links": [
    { "source": "node-1", "target": "node-2", "kind": "related_to" }
  ]
}
```

## Связь с extended-mind-console

| Аспект | extended-mind-console | dream-graph |
|--------|----------------------|-------------|
| **Назначение** | Редактирование | Визуализация |
| **Вкладки** | Редакторы проектов | Сцены рендера |
| **Данные** | Сохраняет сессии | Только читает |
| **Результат** | Cosmos Map JSON | Визуальная сцена |

### Workflow

1. Редактируете таксономию в **Core Taxonomy Editor**
2. Экспортируете как **Cosmos Map JSON**
3. Загружаете в **dream-graph** для визуализации
4. Экспериментируете с разными сценами

## Технологии

- **3d-force-graph** — force-directed layouts
- **Three.js** — WebGL рендеринг
- **Vite** — сборка
