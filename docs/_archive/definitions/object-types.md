# Object Types — Ontology Core v0.1

Этот документ определяет формальные типы объектов в extended-mind и правила их использования.  
Каждый объект принадлежит Мир-3 (объективный артефакт), но может описывать элементы Мира-2.

---

## 1. Project (Проект)
**Определение:**  
Целостный смысловой узел с миссией, границами и собственной архитектурой.

**Характеристики:**
- структура, а не поток мысли;
- имеет связь с целями utemix-lab;
- может включать персонажей, линии, тезисы, story-node;
- имеет собственный README.

**Примеры:**  
`Vova & Petrova`, `evoquant`, `word-machine`, `extended-mind`.

---

## 2. Thesis (Тезис)
**Определение:**  
Минимальная оформленная мысль, выраженная коротко, однозначно, пригодная для векторизации.

**Применение:**  
Строительный материал для линий, концептуальных карт, ADR.

---

## 3. Line / Theme (Линия)
**Определение:**  
Долговременное направление мышления или исследования.

**Признаки:**
- объединяет тезисы,
- проходит через несколько проектов,
- может порождать подпроекты.

---

## 4. Story-node
**Определение:**  
Минимальная фиксация мыслительной динамики: сцена, озарение, тупик, размышление.

**Свойства:**
- относится к Мир-3 *по факту фиксации*;
- может описывать субъективные состояния Мира-2;
- используется как сырьё.

**Подтипы:**
- *Author-line* — личные фрагменты автора;  
- *Machine-log* — фиксации технического процесса (напр. история GitHub).

---

## 5. Character (Персонаж)
**Определение:**  
Функциональный актор Мира-3, выполняющий роль внутри проекта или экосистемы utemix-lab.

**Особенности:**
- не художественная маска;
- имеет обязанности;
- связан с проектами, линиями, тематическими зонами.

---

## 6. ADR (Architectural Decision Record)
**Определение:**  
Структурный документ о принятом архитектурном решении.

**Функция:**  
Создаёт тех-историю принятия решений.

---

## 7. Conceptual Map (Карта смыслов)
**Определение:**  
Структура, отображающая связи между проектами, линиями, тезисами, персонажами и ADR.

---

## 8. Media Unit (Медиа-единица)
**Определение:**  
Публичный носитель смыслов: рилс, ролик, пост, рубрика.

---

## 9. Metaprinciple (Метапринцип)
**Определение:**  
Правило культуры мышления и разработки.

---

## 10. Knowledge Object (Объект знаний)
Общее имя для *любого* файла, пригодного для RAG:
- проект,
- тезис,
- ADR,
- структура,
- определение,
- персонаж,
- карта смыслов.
