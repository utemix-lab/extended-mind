# How to use extended-mind console

Короткая инструкция по использованию Space: поиск, режимы вывода и экспорт.

## Поиск
- Введите запрос и нажмите **Search**.
- Настройте `Top K`, чтобы увеличить или уменьшить число результатов.

## Режимы вывода
- **Cards** — карточки с текстом фрагмента.
- **Table** — компактная таблица: score, путь, заголовок, превью.

## Фильтры и шум
- **Min score** — минимальный порог релевантности.
- **Deduplicate by rel_path** — убирает дубликаты по одному пути.
- **Prefer manifest/adr for "what is"** — при запросах "what is/что такое" чуть поднимает manifest/adr.
- **Show full text (cards)** — показывать полный текст в карточках.

## Экспорт
- **Export JSON** — выводит JSON результатов (удобно копировать как код).
- **Copy bundle** — формирует компактный “пакет” текста: вопрос + список фрагментов с путями.
