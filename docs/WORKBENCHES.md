# WORKBENCHES

Типы workbench’ей (без привязки к конкретным персонажам).

## Формат (минимум)

- **ID**
- **Цель**
- **Вход**
- **Выход**
- **Риски**

## Типы (черновик)

### workbench‑signals‑scan
- **Цель:** быстрый обзор новых сигналов
- **Вход:** каталоги/ссылки/теги
- **Выход:** shortlist + story‑node
- **Риски:** медийный захват, каталогизация

### workbench‑tool‑review
- **Цель:** разбор новых инструментов/сервисов
- **Вход:** tool cards, docs, demos
- **Выход:** практика + следы в каталоге
- **Риски:** “второй интернет”, переобъективация

### workbench‑route‑craft
- **Цель:** сбор маршрутов и траекторий
- **Вход:** контекст‑стек, практики, query‑результаты
- **Выход:** saved path + story‑node
- **Риски:** фиксация без смысла
