# Workbench: As if

**ID:** wb-as-if  
**Characters:** Вова, Вася  
**Domain focus:** Music & Sound / Visual Media
**Type:** collaborative / in-world production
**Slogan:** —

## Purpose
Создание авторской музыки и саунд-дизайна для самой системы.

## Focus
- Музыка для видео-контента системы
- Закадровые треки
- Атмосферные и жанровые эксперименты
- Саунд как часть айдентики

## Inputs
- Темы и настроения дайджестов
- Визуальные сцены системы
- Эксперименты Вовы и Васи

## Practices
- `practice-direction` (см. `docs/specs/PRACTICES_SPEC.md`)
- `practice-sound-thinking` (см. `docs/specs/PRACTICES_SPEC.md`)
- `practice-visual-thinking` (см. `docs/specs/PRACTICES_SPEC.md`)

## Outputs
- curated_sets.jsonl (треки и подборки As if)
- signals.jsonl (новые релизы внутри системы)
- аудио-ассеты для контента

## Preset Context

domain:music
domain:visual
practice:direction

## Boundaries
- Не музыкальный лейбл.
- Не каталог всей музыки.
- Работает только на нужды системы и её эстетики.

## UI Reflection (Visitor)
- **Story:** Образы группы As if, клипы, сцены.
- **Service:** Прослушивание, привязка к контенту.
- **System:** Связь с айдентикой и брендом.

## Risks
- Размывание фокуса между проектом и системой.
- Превращение в отдельный “музыкальный проект”.
