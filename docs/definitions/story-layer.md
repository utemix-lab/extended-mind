# Story Layer — Definition  
Версия 0.1

---

## 1. Что такое story-layer

Story-layer = совокупность story-node, author-line и machine-log.  
Это “исторический слой” — динамика мышления.

---

## 2. Функции story-layer

- сохраняет ход мысли,  
- фиксирует ошибки и озарения,  
- служит сырьём для тезисов,  
- позволяет воспроизводить историю проектов.

---

## 3. Что story-layer не является

- структурой,  
- архитектурой,  
- решением,  
- проектным документом.

---

## 4. Место story-layer в системе слоёв

Story → Theses → Lines → Concept Map → Architecture → Media.

---

## 5. Правила для extended-mind

- не смешивать story-layer с инженерным слоем,  
- не использовать story-node напрямую как решение,  
- хранить историю без удаления.
