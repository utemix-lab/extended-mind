# Capability Map — Tags & Query Mode

## Контекст этапа
Этап завершает внедрение pointer-tags и Query Mode как базового механизма
ориентации в домене AI без расширения канона. Он закрепляет практику работы
с каталогами и режимами запроса вместо страниц и онтологических сущностей.

## Архитектурные находки
- Tag as query anchor
- Query Mode as default domain state
- Canon ≠ Catalog
- Perception ≠ Ontology

## Practices
- practice-direction
