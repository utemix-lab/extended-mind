# Pattern: Line Consolidation  
Версия 0.1

---

## 1. Проблема

В процессе развития extended-mind возникают:

- несколько линий,  
- близких по теме,  
- пересекающихся по функциям.

Слишком много мелких линий → потеря структуры.

---

## 2. Решение

Если две или более линии:

- имеют один центр смысла,  
- являются ответами на один вопрос,  
- не противоречат друг другу,

их можно **консолидировать**.

---

## 3. Шаги

1. Сравнить цели линий.  
2. Проверить совпадение ключевых тезисов.  
3. Определить доминирующий смысл.  
4. Создать новую линию “второго уровня”.  
5. Старые линии пометить как “слиты”.

---

## 4. Результат

Появляется более мощная линия:  
- глубже,  
- устойчивее,  
- структурно важнее.
