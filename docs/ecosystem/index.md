📖 **Философия экосистемы:** [VISION.md](../../VISION.md)

```
┌─────────────────────────────────────────────────────────────────────┐

└─────────────────────────────────────────────────────────────────────┘
                              ↓ контент, методы, идеи

│            (мета-координатор, RAG, Universe Graph)                  │

         ↓                        ↓                        ↓
┌────────────────┐   ┌────────────────────┐   ┌────────────────────────┐
│ vovaipetrova-  │   │ extended-mind-     │   │   minds-lorgnette      │
│     core       │   │    console         │   │ (универсальное         │
│   (канон)      │   │  (редакторы)       │   │   приложение)          │
└────────────────┘   └────────────────────┘   └────────────────────────┘
         ↓                        ↓
┌────────────────┐   ┌────────────────────┐
│ vovaipetrova-  │   │    dream-graph     │
```

## Проекты экосистемы

| Проект | Тип | Роль |
|--------|-----|------|
| **extended-mind** | Ядро | Мета-координатор, RAG, Universe Graph |
| **extended-mind-console** | Инструмент | Редакторы таксономий |
| **vovaipetrova-core** | Контент | Канон (read-only источник) |
| **vovaipetrova-site** | Инструмент | Классический web-интерфейс |
| **dream-graph** | Инструмент | 3D визуализация графов |
| **minds-lorgnette** | Производная | Универсальное приложение для знаний |
| **evoquant** | Исследование | Пища для мышления |

## Типы проектов

### Ядро: extended-mind

Мета-координатор всей экосистемы:
- Universe Graph — единый граф связей
- RAG — поиск по базам знаний
- Архитектурные решения (ADR)
- Координация между проектами

### Инструменты

| Инструмент | Назначение | Вкладки |
|------------|------------|---------|
| **extended-mind-console** | Редактирование | Console (RAG), Core Taxonomy Editor |
| **dream-graph** | Визуализация 3D | Planetary, Mono, Universe, Core Taxonomy |
| **vovaipetrova-site** | Классический UI | Layouts, Styles, Components |

### Контент: vovaipetrova-core

- Канон — неизменяемый источник истины
- KB (термины), Stories (эпизоды)
- Статический контракт (routes, tokens)
- **Read-only** для всех инструментов

### Производные: minds-lorgnette

Расширение extended-mind за пределы экосистемы:
- Интеграция способов организации знаний (таблицы, графы, майнд-карты, канбаны)
- Интеграция с Obsidian, Notion, Roam
- Универсальное приложение

### Исследования: evoquant

Особый тип проектов (таких будет несколько):
- Исследование способов мышления
- Границы академических наук, мифов, религиозных доктрин
- Генерация контента для vovaipetrova-core
- Формирование методов для extended-mind

## Потоки данных

### Контент

```
evoquant (исследования)
        ↓ контент, термины, истории
vovaipetrova-core (канон)
        ↓ read-only данные
vovaipetrova-site / dream-graph (публикация)
```

### Методы

```
evoquant (эксперименты)
        ↓ паттерны, методы
extended-mind (систематизация)
        ↓ архитектура
minds-lorgnette (расширение)
```

### Визуализация

```
extended-mind-console (редактирование)
        ↓ Cosmos Map
dream-graph (3D рендер)
        ↑
vovaipetrova-site (классические UI-элементы)
```

## Инструменты как среды со вкладками

Все инструменты экосистемы организованы как **среды со вкладками**:

### extended-mind-console

| Вкладка | Назначение | Статус |
|---------|------------|--------|
| Console | RAG-среда (эксперимент) | Активна |
| Core Taxonomy Editor | Редактор vovaipetrova-core | Активен |
| *(будущие)* | Редакторы других проектов | Планируется |

### dream-graph

| Сцена | Назначение |
|-------|------------|
| Planetary | Орбитальная модель |
| Mono Graph | Минималистичная сцена |
| Universe Map | Экосистемный вид |
| Core Taxonomy | Граф vovaipetrova-core |

### vovaipetrova-site

| Раздел | Назначение |
|--------|------------|
| Layouts | HTML шаблоны |
| Styles | CSS, дизайн-токены |
| Components | UI-компоненты |
| Core | Навигация, роутинг |
