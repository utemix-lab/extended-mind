# ROLES OF THOUGHT

Роль = «режим обработки смыслов».

Extended-mind использует ролевое мышление как методику чередования
когнитивных стратегий.

---

# 1. Core Roles

## 1.1. Negative Engineer  
Ищет слабые места, уязвимости, провалы.

Форма вопроса:  
*«Почему это не будет работать?»*

## 1.2. Architect  
Создаёт структуры, уровни, схемы.

Форма вопроса:  
*«Какие формы соответствуют этому смыслу?»*

## 1.3. Cognitive Mapper  
Соединяет сущности, строит карту связей.

Форма вопроса:  
*«Что с чем связано?»*

## 1.4. Fiction Engine  
Генерирует художественные интерпретации.

Форма вопроса:  
*«Как это выглядит в образах, сюжетах, настроениях?»*

## 1.5. Compression Engine  
Сжимает смысл в текст, summary, embedding-кандидаты.

Форма вопроса:  
*«Как это выразить короче?»*

---

# 2. Supporting Roles

## 2.1. Chaos Agent  
Вбрасывает аномалии, ломает шаблоны.

## 2.2. Synthesis Engine  
Собирает устойчивую форму из противоречий.

## 2.3. Historian  
Удерживает связь с предыдущими версиями смысла.

---

# 3. Правила ролевого переключения

Роли включаются:
1. Последовательно  
2. По мере сложности задачи  
3. С учётом Layer of Impossibilities

Типичная цепочка:

Negative Engineer → Architect → Mapper → Fiction Engine → Compression
